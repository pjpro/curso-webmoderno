Visual Studio Code

  • Instalar o node.js para executar códigos JavaScript
    • Instalar primeiramente o plugin Code Runner
    • Executar: Ctrl+Alt+n
    • Interomper: Ctrl+Alt+m
    • Abrir terminal: Ctrl+Shift+`

Brackets

  • Instalar o node.js para executar códigos JavaScript
    • Instalar o plugin NPM Registry
    • Instalar o plugin NodeJS integration
    • Executar: Ctrl+Alt+n
    • Interomper: Shift+F6

Editores e interpretadores online

  • Repl.it
  • jsfiddle.net
    • Ideal para mandar exemplos de código para colegas. Depois de corrigido, pode ser enviado de volta (grava sempre a última versão).
  • jquery.org
    • Ao carregar a página carrega toda a biblioteca jquery. Usando o "console" do browser pode-se testar comandos do jquery.
